Berikut adalah penjelasan cara penggunaan kode untuk lazy loading dengan animasi saat berpindah halaman:

### 1. Struktur File
Anda akan memiliki dua file HTML:
- `lazy_load.html`
- `lazy_load2.html`

### 2. Kode `lazy_load.html`
- **HTML Dasar**: Memuat Bootstrap dan jQuery untuk styling dan fungsionalitas.
- **Loader**: Terdapat elemen `<div id="loader">` yang berisi spinner dan pesan "Loading...". Secara default, elemen ini disembunyikan dengan `display: none;`.
- **Tombol Pindah Halaman**: Tautan ke `lazy_load2.html` dengan ID `loadPage`.

### 3. Kode JavaScript
- **jQuery**: Setelah halaman dimuat (`$(document).ready`), event handler ditambahkan pada tombol `loadPage`.
- **Prevent Default**: Menggunakan `e.preventDefault()` untuk mencegah perilaku default tautan (pindah halaman langsung).
- **Tampilkan Loader**: Ketika tombol diklik, elemen loader ditampilkan dengan `$('#loader').show();`.
- **Delay Sebelum Pindah**: Menggunakan `setTimeout()` untuk menunggu 1 detik (1000 ms) sebelum mengalihkan halaman menggunakan `window.location.href`.

### 4. Kode `lazy_load2.html`
- Memuat tampilan untuk halaman kedua dengan tombol kembali ke `lazy_load.html`.

### 5. Cara Menggunakan
1. **Membuat File**: Buat dua file HTML dengan nama yang sesuai.
2. **Salin Kode**: Salin kode dari masing-masing contoh di atas ke file yang sesuai.
3. **Buka di Browser**: Buka `lazy_load.html` di browser Anda.
4. **Klik Tombol**: Klik tombol "Pindah ke Halaman 2". Anda akan melihat animasi loading sebelum berpindah ke `lazy_load2.html`.
5. **Kembali**: Gunakan tombol di halaman kedua untuk kembali ke halaman pertama.

### 6. Penyesuaian
- **Waktu Delay**: Anda dapat mengubah durasi delay pada `setTimeout()` jika ingin memperpanjang atau memperpendek waktu loading.
- **Tampilan Loader**: Anda bisa mengubah gaya CSS pada loader sesuai keinginan untuk menciptakan efek loading yang berbeda.
